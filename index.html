About Kodi

Hi! I'm Kodi, a software engineer residing in the San Francisco, Bay Area.

Along with my software skills mentioned above, I have an extensive background in education and travel, with a special knack for finding adventure. Since I've been fortunate enough to live and teach in three different countries - Japan, China, and the USA - I’ve grown accustomed to working in diverse environments and building unique global perspectives. Looking forward to working with open-minded and creative teams to help make this world better for future generations.

Recent Projects

Fakebook

This is a clone of the popular social media application Facebook. For this project I used a Ruby on Rails backend and JavaScript with React and Redux for the frontend. I incorporated Amazon Web Services S3 for uploading photos.

WeHaul

WeHaul is a unique MERN (MongoDB, Express, React, Node) Stack project that is mixture of Lyft and Uhaul. Google Maps API along with Mapbox API are used to find addresses in the SF region and map the routes.

Color Breaker

This fun game uses pure JavaScript for the game logic combined with CSS3 and HTML 5 canvas for design and animation. A simple UI makes this game both fun and intense. Try to play on insane mode and see how long you can last!
